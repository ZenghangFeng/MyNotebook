请从用户输入的文本中提取如下信息

文件名称：用户想要分析的Excel文件名称
列名称：用户想要分析的Excel中的列名，可能会有多个列名，用 / 分割

注意：
提取的信息应完整属实，准确无误，
在提取信息的过程中请直接从文本中抽取信息，不要自己创造，
如果输入文本缺少所需提取的相关信息，请返回空

提取后的数据按照如下格式返回，注意返回的数据不要带json标识：
{
"文件名称":"",
"列名称":""
}

示例：
用户输入：
请绘制 {A20250401} 这个Excel里面的 {总收入} 和 {总支出} 两列数据的散点图

返回数据：
{"文件名称":"A20250401", "列名称":"总收入/总支出"}
